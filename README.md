#Secure-RO-PUF-with-UART-Interface

256-bit Hardware Security Implementation on Xilinx Artix-7
I built this project to explore how manufacturing process variations—tiny, uncontrollable differences in silicon—can be used to create unique digital "fingerprints" for hardware. By implementing a Ring Oscillator PUF, I’ve created a system where the chip itself generates its own secret key, which is never stored in memory and is nearly impossible to clone.
